# MeshCore-MQTT-WebFlasher

Standalone web flasher and serial configurator for MeshCore MQTT repeater firmware.

This project provides a browser-based firmware flasher and post-flash serial setup tool
for MeshCore MQTT repeater builds. It is designed to flash supported boards, apply
radio, device, WiFi, and MQTT settings over Web Serial, and verify the resulting device
state from the same web interface.
